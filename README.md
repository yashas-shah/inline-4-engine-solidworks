# Inline 4-Cylinder Engine Assembly (SolidWorks)

##  Overview
This project involves the design and assembly of a complete inline 4-cylinder engine using SolidWorks.  
The model demonstrates how reciprocating motion from pistons is converted into rotational motion through the crankshaft, showcasing the interaction between key engine components.

---

##  Components Included
- Pistons (4 cylinders)
- Connecting Rods
- Crankshaft
- Piston Pins
- Cylinder Block (if included)
- Complete Engine Assembly
